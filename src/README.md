## This is a cron job expression parser

##### To run this code, we can intellij or terminal to run this code

##### Using terminal one can use this to compile the code: javac ./constants/*.java ./exceptions/*.java ./factory/*.java ./factory/implementation/*.java ./models/*.java ./parser/*.java ./response/*.java ./parser/implementation/*.java ./utils/*.java Main.java

##### To execute the command please use java Main "*/15 0 1,15 * 1-5 /usr/bin/find" 

##### When using intellij, go to Edit Configuration and add program arguments to run this script directly.
